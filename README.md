### 📄 Sobre el proyecto 

Tienda de Joyas personalizables, usando Front-end, "React", "Bootstrap" y usando Back-end "EmailJs".

### 🌐 Despliegue

Tan sólo deberás de instalar las librerías requeridas con **npm install**. y darle a **npm start**
